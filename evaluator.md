# Volunteer Evaluator — Feedback & Continuous Improvement

## Deutsch

### Was ist das?
Ein einfaches Feedback-Formular für Volunteers nach der Nutzung des 
Chat-OP Decision Assistenten. Ziel: strukturiertes Feedback sammeln 
und direkt als neue Edge Case Rulings in den Assistenten einbauen.

**KI empfiehlt — Mensch entscheidet. Immer.**

### Wie funktioniert es?
1. Volunteer nutzt den Chat-OP Decision Assistenten
2. Volunteer füllt das Feedback-Formular aus
3. Nathalie/Carmen wertet das Feedback aus
4. Neue Edge Cases fließen direkt in CLAUDE_Nathi.md oder 
   CLAUDE_Cara.md ein
5. Aktualisierte Datei wird ins Repository hochgeladen

### Was wird abgefragt?
| Feld | Beschreibung |
|---|---|
| Fall-ID | Referenz zum gemeldeten Fall |
| Klassifizierung korrekt? | Ja / Nein / Teilweise |
| Warnungsentwurf nutzbar? | Ja / Nein / Angepasst |
| Grenzfall? | Ja / Nein |
| Beschreibung | Was war unklar oder falsch? |
| Vorschlag | Wie sollte es klassifiziert werden? |

### Wichtig
- Kein Feedback wird automatisch übernommen
- Alle Änderungen werden von Nathalie oder Carmen geprüft
- Human-in-the-loop gilt auch für die Weiterentwicklung

## English

### What is it?
A simple feedback form for volunteers after using the Chat-OP 
Decision Assistant. Goal: collect structured feedback and feed it 
directly into new Edge Case Rulings for the assistant.

**AI recommends — human decides. Always.**

### How does it work?
1. Volunteer uses the Chat-OP Decision Assistant
2. Volunteer fills in the feedback form
3. Nathalie/Carmen reviews the feedback
4. New edge cases are added directly to CLAUDE_Nathi.md or 
   CLAUDE_Cara.md
5. Updated file is uploaded to the repository

### What is collected?
| Field | Description |
|---|---|
| Case ID | Reference to the reported case |
| Classification correct? | Yes / No / Partially |
| Warning draft usable? | Yes / No / Adapted |
| Edge case? | Yes / No |
| Description | What was unclear or wrong? |
| Suggestion | How should it have been classified? |

### Important
- No feedback is applied automatically
- All changes are reviewed by Nathalie or Carmen
- Human-in-the-loop applies to further development too
